# LegalAI Platform - Plataforma Jurídica Unificada com IA

## 1. Visão Geral

A LegalAI Platform é uma solução completa e integrada que implementa todas as 8 recomendações de integração de IA para o setor jurídico. A plataforma oferece um conjunto de ferramentas avançadas para otimizar a rotina de advogados e escritórios, desde a gestão de processos até a análise de dados e geração de documentos.

## 2. Funcionalidades Implementadas

A plataforma implementa as seguintes funcionalidades, cada uma correspondendo a uma recomendação de IA:

1.  **Análise Inteligente de Documentos:** Upload e análise automática de documentos jurídicos com IA, extraindo informações, identificando conceitos e fornecendo recomendações.
2.  **Automação de Workflows:** Criação e gerenciamento de fluxos de trabalho personalizáveis para automatizar tarefas repetitivas.
3.  **Assistente Virtual Jurídico:** Assistente inteligente para consultas e orientações jurídicas em tempo real.
4.  **Gestão Inteligente de Prazos:** Monitoramento automático e alertas de prazos processuais, evitando perdas de prazo.
5.  **Analytics Jurídico:** Análise de dados processuais, métricas de performance e insights estratégicos para tomada de decisão.
6.  **Geração de Documentos:** Geração automática de petições, contratos e outros documentos jurídicos usando templates e IA.
7.  **Pesquisa Inteligente:** Busca avançada em jurisprudência, legislação e doutrina com IA e processamento de linguagem natural.
8.  **Cálculos Jurídicos Avançados:** Calculadora completa para cálculos trabalhistas, cíveis, atualizações monetárias e juros.

## 3. Arquitetura e Tecnologias

A plataforma foi desenvolvida com uma arquitetura moderna e escalável, utilizando as seguintes tecnologias:

*   **Frontend:** React, Vite, Tailwind CSS, shadcn/ui, Lucide Icons, Recharts
*   **Backend:** Python, Flask (para os serviços de IA)
*   **Inteligência Artificial:** OpenAI GPT-4, Scikit-learn, Pandas, NumPy

## 4. Como Executar a Aplicação

Para executar a aplicação em um ambiente de desenvolvimento, siga os passos abaixo:

1.  **Clone o repositório:**

    ```bash
    git clone <URL_DO_REPOSITORIO>
    cd legal-ai-unified-platform
    ```

2.  **Instale as dependências:**

    ```bash
    pnpm install
    ```

3.  **Inicie o servidor de desenvolvimento:**

    ```bash
    pnpm run dev --host
    ```

4.  **Acesse a aplicação:**

    Abra o navegador e acesse `http://localhost:5174`.
